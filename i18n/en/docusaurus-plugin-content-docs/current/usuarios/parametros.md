Pick a flag to see details and examples:

- `--target-url` (`-t`): see [Target URL](./target-url)
- `--api-url` (`-u`): see [Server URL](./api-url)
- `--add-cors-headers` (`-c`): see [CORS](./add-cors-headers)
- `--extra-header` (`-e`): see [Custom headers](./extra-header)
- `--mock-config` (`-m`): see [Mocks](./mock-config)
- `--save-request-directory` (`-s`): see [Save responses](./save-request-directory)
- `--show-headers` (`-h`): see [Show headers](./show-headers)
- `--show-body` (`-b`): see [Show body](./show-body)

Minimal example
- ```bash
  proxxyy -t 'https://api.example.com' -u 'http://localhost:6969'
  ```
