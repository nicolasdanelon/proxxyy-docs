Selecciona un flag para ver detalles y ejemplos:

- `--target-url` (`-t`): ver [Target URL](./target-url)
- `--api-url` (`-u`): ver [API URL](./api-url)
- `--add-cors-headers` (`-c`): ver [CORS](./add-cors-headers)
- `--extra-header` (`-e`): ver [Cabeceras extra](./extra-header)
- `--mock-config` (`-m`): ver [Mocks](./mock-config)
- `--save-request-directory` (`-s`): ver [Guardado de respuestas](./save-request-directory)
- `--show-headers` (`-h`): ver [Mostrar cabeceras](./show-headers)
- `--show-body` (`-b`): ver [Mostrar cuerpo](./show-body)

Ejemplos mínimos
- ```bash
  proxxyy -t 'https://api.example.com' -u 'http://localhost:6969'
  ```
